# Friends Table – SQL Table Operations

SQL exercise from Codecademy practicing basic table manipulation.

Concepts practiced:
- creating tables (CREATE TABLE)
- inserting records (INSERT)
- updating data (UPDATE)
- modifying table structure (ALTER TABLE)
- deleting records (DELETE)
- querying data (SELECT)

The script creates a simple table of friends, inserts sample data,
adds a new column for email addresses, updates records, removes
one entry, and displays the remaining records.
