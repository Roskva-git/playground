# Playground

This repository contains small projects and exercises I've done to learn about programming.  
Most of them are from Codecademy.

The goal of this repository is to document concepts I have practiced across different languages and tools, including:

- Python
- R
- SQL
- JavaScript

Each folder contains a small self-contained exercise focused on a particular concept (e.g., control flow, data manipulation, SQL aggregation, or basic scripting).
